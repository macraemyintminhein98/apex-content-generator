# IdeaSpark

IdeaSpark is a simple, AI-powered web application designed to help business owners, content creators, and marketers overcome content block. 

**Features:**
*   Generate blog post topics, social media captions, or video ideas.
*   Input your keyword/niche and get instant suggestions.
*   One-time purchase for ongoing access.

**How it works (Behind the Scenes):**
1.  User enters a keyword.
2.  A crafted prompt is sent to a fine-tuned AI model (e.g., GPT-3.5 turbo via OpenAI API).
3.  The AI generates relevant content ideas.
4.  Ideas are displayed to the user.

This project focuses on core functionality and a minimal, intuitive interface. Designed for quick setup and launch.
